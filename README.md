## Overview
This repository contains the required experiments, labs, and code samples for the Infrastructure & Cloud course (DevNet Associate).

## Repository Structure
The project is organized by topic, corresponding to the evaluation criteria:

* **Data_Formats/**: Parsing JSON/YAML and data model conversion (Tasks Js1-Js4, Yg1).
* **API/**: Python and cURL scripts for Webex, RESTCONF, and custom APIs (Tasks Ap1-Ap6).
* **NetConf_RestConf/**: Network programmability using Netmiko, RESTCONF, and NETCONF (Tasks N1-N4).
* **DNAC/**: Cisco DNA Center Sandbox interaction scripts (Task Dna).
* **Ansible/**: Playbooks for IOS XE routers and Linux servers (Tasks An1-An3, As1-As3).
* **Flask_Microservices/**: Python Flask web applications (Tasks Pf1-Pf3).
* **Docker/**: Dockerfiles and container management scripts (Tasks Di1-Di3, Dc1, Dm1).
* **Jenkins/**: CI/CD pipeline configurations (Tasks J1-J2).
* **Webex/**: Webex automation scripts (Tasks W1-W2).
* **Venv_Bash/**: Virtual Environment setup and Bash automation (Tasks Pv1, Bash).
