# Unit Testing

## Tasks Overview
This folder contains the Unit Testing experiment, satisfying criteria U1 (Lab 3.5.7).

### Included Files
* **`recursive_json_search.py`**: A Python function that searches for a specific key within a nested JSON structure (Dict/List).
* **`test_json_search.py`**: A `unittest` script that verifies the search function works correctly for nested, list, and missing keys.

### Usage
